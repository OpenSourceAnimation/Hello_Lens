Hello_Lens
HoloLens + Unity Intro
Nick Fox-Gieg

<< Nick’s site  |  repo >>  |  etherpad >>

( 16 Nov 17, Thu.  10am–1pm )

  1. Which version of Windows 10 do I need? The repo examples were made on Windows 10 Pro Fall Creators Edition (1709). Check your Windows version with Settings > System > About.

  2. Which version of Visual Studio do I need? You want Visual Studio 2017, the Universal Windows Platform (UWP) build tools, and the Windows 10.0.16299 SDK. Check that these options are installed with Tools > Get Tools and Features.

  3. Which version of Unity do I need? You want Unity 2017.2, currently the newest version of Unity. When installing, add both of the two UWP build options.

  4. Where should I start? Try this simple hand tracking example as a base for your first project.

  5. Do I need to use the Mixed Reality Toolkit Unity package? No. It does have useful examples, but the core interaction functionality is now included in Unity.

  6. Which version of the Mixed Reality Toolkit should I use? Grab the master repo, because currently the last release version isn’t ready for Unity 2017.2.

  7. Is the HoloToolkit Unity package the same thing as the Mixed Reality Toolkit? Yes. The two names are used interchangeably throughout the HoloLens documentation. They’re the same thing.

  8. Do I need to install the standalone HoloLens Emulator? No. It now comes included with recent versions of Unity.
